# Personal Portfolio Website

This project is a **personal portfolio website** built using **HTML5 and CSS3**.  
It showcases personal information, featured projects, and a contact section, with a clean and consistent visual identity.

The main goal of this project is to practice **layout design, responsive structure, and visual coherence** using modern CSS techniques such as **Grid, Flexbox, and aspect-ratio**, without relying on JavaScript frameworks.
